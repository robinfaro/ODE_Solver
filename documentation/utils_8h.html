double ParseFraction ( const std::string & fraction)

Parses a string potentially representing a fraction and returns the result as a double.

The function parses a string representing a fraction and returns the result as a double. The fraction should be in the form "numerator/denominator". If the string does not contain a '/', the function will attempt to parse the string as a plain number.

Parameters

Returns
double The result of the fraction as a double.
