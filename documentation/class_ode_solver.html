◆ Solve()

virtual Eigen::MatrixXd OdeSolver::Solve ( )
pure virtual

Solve the ODE problem.

This is a pure virtual function that must be implemented by derived classes. It solves the ordinary differential equation over the specified time interval with the given initial conditions and step size.

Returns
Eigen::MatrixXd A matrix containing the solution of the ODE at each time step.
