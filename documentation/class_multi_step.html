Detailed Description

A class for solving ordinary differential equations (ODEs) using multi-step methods.

Multi-step methods are solvers of ordinary differential equations (ODEs) which utilize information from multiple previous steps to compute the next solution point. They are generally written in the form:

\[ y_{n+1} = \sum_{i=0}^{k} \alpha_i y_{n-i} + h \sum_{i=0}^{k + 1} \beta_i f(t_{n-i}, y_{n-i}), \]

where:

  • \( y_{n+1} \): the solution at the next time step.
  • \( y_{n-i} \): the solutions at previous time steps.
  • \( f(t, y) \): the derivative of the function \( \frac{dy}{dt} \).
  • \( \alpha_i, \beta_i \): coefficients that change according to the multi-step method employed.
  • \( h \): the time step size.
  • \( k \): the number of previous steps used in the method.
